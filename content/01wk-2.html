1. R은 좋은 계산기이다.

  1. \(-5\)

  3. \(0\)
  4. \(\sqrt{5}\)
  5. \(5\)

4. 함수의 선언

- 함수의 기본 구조

함수이름 <- function(입력){
    함수의 기능 
    return(출력)
    }
solve <- function(input){
    theta = acos(input) - pi/2
    output = sin(theta)/(1-cos(theta)^2)
    return(output)
}
solve(-1/5)
[1] 5

- 아래의 2개의 코드는 거의 같다.

### 1 
solve <- function(input){
    theta = acos(input) - pi/2
    output = sin(theta)/(1-cos(theta)^2)
    return(output)
}
solve(-1/5)

### 2
input = -1/5 
theta = acos(input) - pi/2
output = sin(theta)/(1-cos(theta)^2)
output
[1] 5

- return을 생략해도 괜찮다.

solve <- function(input){
    theta = acos(input) - pi/2
    output = sin(theta)/(1-cos(theta)^2)
    return(output)
}
solve(-1/5)
[1] 5
solve <- function(input){
    theta = acos(input) - pi/2
    output = sin(theta)/(1-cos(theta)^2)
    output
}
solve(-1/5)
[1] 5
solve <- function(input){
    theta = acos(input) - pi/2
    sin(theta)/(1-cos(theta)^2)
}
solve(-1/5)
[1] 5
