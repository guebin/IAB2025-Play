기본정보

Nationality 국적
Flag 국가의 국기 이미지 (URL 또는 이미지 경로)
Club 소속 클럽

Wage 주급 (예: €200K 등)
Release Clause 바이아웃 금액, 계약 해지에 필요한 금액

컬럼명 설명
Overall 현재 종합 능력치 (게임 내 평점 기준)

Best Overall Rating 가장 높은 능력을 발휘할 수 있는 포지션에서의 최고 평점
Best Position 가장 잘하는 포지션

컬럼명 설명

Body Type 체형 (예: Lean, Normal, Stocky)

Position 주 포지션 (예: ST, CM, GK 등)
Jersey Number 등번호
Joined 클럽에 입단한 날짜
Loaned From 임대 중인 경우 원 소속 클럽
Contract Valid Until 계약 만료 연도

컬럼명 설명
Height 키 (cm 또는 ft/in)
Weight 몸무게 (kg 또는 lbs)

df = pd.read_csv('FIFA22_official_data.csv').drop(["Loaned From", "Marking"], axis=1).dropna()
df['Position'] = [l.split(">")[-1] for l in df.Position]
position_dict = {
    'GOALKEEPER':{'GK'},
    'DEFENDER':{'CB','RCB','LCB','RB','LB','RWB','LWB'},
    'MIDFIELDER':{'CM','RCM','LCM','CDM','RDM','LDM','CAM','RAM','LAM','RM','LM'},
    'FORWARD':{'ST','CF','RF','LF','RW','LW','RS','LS'},
    'SUB':{'SUB'},
    'RES':{'RES'}
}
df['Position_Group'] = [[k for k,v in position_dict.items() if l in v].pop() for l in df.Position]
df['Wage'] = [eval(l.replace("€","").replace("K","*1000").replace("M","*1000000")) for l in df.Wage]
df_defender_only = df[df.Position_Group == "DEFENDER"].copy()
df_defender_only['SlidingTackleCut'] = pd.qcut(df_defender_only['SlidingTackle'], q=4)
df_defender_only.groupby("SlidingTackleCut")['Wage'].mean()
