# 예제 – 아래의 벡터를 고려하자.

set.seed(43052)
x = rnorm(100)

(1) x의 원소중 그 값이 양수인 것은 모두 몇개인가?

sum(x>0)
[1] 59

(2) x의 평균을 구하여라.

mean(x)
[1] 0.1044026

(3) x의 원소중 평균과 가장 가까운 값을 구하여라.

(풀이1)

dist = abs(x - mean(x)) 
# dist는 x와 x의 평균간의 차이를 나타내는 벡터
tf_array <- (dist == min(dist))
# tf_array 는 길이가 100이고 true, false로만 구성된 array 
# 그런데 이 tf_array는 하나의 true값만 있고, 그 true값은 dist가 가장 작은 지점에서 true를 가짐
x[tf_array]
[1] 0.102093

(풀이2)

idx = which.min(dist)
# idx 는 dist의 최소값이 존재하는 위치(index)가 저장되어 있음
x[idx]
[1] 0.102093

새로운 문제 – dist에서 which.min을 쓰지 않고 위에서 구한 idx를 얻는 방법?

2007(나)6월평가원-14. 다음은 어느 회사의 연봉에 대한 규정이다.

(가) 입사 첫째 해 연봉은 \(a\)원이고, 입사 19년째 해까지의 연봉은 해마다 직전 연봉에서 8%씩 인상된다.

(나) 입사 20년째 해부터의 연봉은 입사 19년째 해 연봉의 \(2/3\)로 한다.

이 회사에 입사한 사람이 28년 동안 근무하여 받는 연봉의 총합은? (단, \(1.08^{18} = 4\)로 계산한다.)

  1. \(\frac{101}{2} a\)
  2. \(\frac{111}{2} a\)
  3. \(\frac{121}{2} a\)
  4. \(\frac{131}{2} a\)
  5. \(\frac{141}{2} a\)

(풀이)

a = 1
sal = c()
sal[1] = a
for(y in 2:19){
    sal[y] <- sal[y-1] * 1.08 
}
for(y in 20:28){
    sal[y] <- sal[19] * 2/3 
}
sum(sal)
[1] 65.42238

2. 어떤 회사에서는 연봉이 매년 일정한 규칙에 따라 인상된다.

  1. 입사 첫해 연봉은 4000만 원이다.
  2. 다음 해부터는 전년도 연봉의 8%를 인상한 후, 추가로 100만 원을 더해 결정한다.

즉, 연봉을 \(a_n\)이라 하면 다음과 같은 관계식을 따른다. (단위는 만원)

\[ \begin{cases} a_1 = 4000 & \\ a_n = a_{n-1} \times 1.08 + 100 & n=2,3,\dots \end{cases} \]

이 회사에서 20년 동안 근무했을 때, 받는 총 연봉의 합은 얼마인가?

(풀이)

a = c()
a[1] = 4000 
for(n in 2:20){
    a[n] = a[n-1] * 1.08 + 100 
}
sum(a)
[1] 215250.3
